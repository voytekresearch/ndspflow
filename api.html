    Workflows¶

    Create a nipype workflow connecting input, FOOOF and Bycycle nodes.

    create_workflow(input_dir, output_dir[, …])

    Connects nodes into a nipype workflow.

    wf_fooof(fooof_params)

    Create the fooof workflow.

    wf_bycycle(bycycle_params)

    Create the bycycle workflow.

    Interfaces¶

    Input, output, and runtime definitions for the FOOOF and Bycycle nodes.

    FOOOFNodeInputSpec(**kwargs)

    Input interface for FOOOF.

    FOOOFNodeOutputSpec(**kwargs)

    Output interface for FOOOF.

    FOOOFNode([from_file, resource_monitor])

    Interface wrapper for FOOOF.

    BycycleNodeInputSpec(**kwargs)

    Input interface for bycycle.

    BycycleNodeOutputSpec(**kwargs)

    Output interface for bycycle.

    BycycleNode([from_file, resource_monitor])

    Interface wrapper for bycycle.

    Model Fitting¶

    Fits FOOOF and Bycycle models for 1D, 2D or 3D arrays.

    fit_fooof(freqs, powers, freq_range, …)

    A generalized FOOOF fit function to handle 1d, 2d, or 3d arrays.

    fit_bycycle(sig, fs, f_range[, …])

    A generalized Bycycle compute_features function to handle 1d, 2d, or 3d arrays.

    Plotting¶

    Create FOOOF and Bycycle plots that will be embedded in html reports.

    plot_fm(fm[, log_freqs, fill_gaussians, …])

    Plot a single FOOOF fits using plotly.

    plot_fg(fg, urls)

    Plot FOOOFGroup parameters distributions using plotly.

    plot_fgs(fgs, urls)

    Plot a list of FOOOFGroup parameters distributions using plotly.

    plot_bm(df_features, sig, fs, …[, xlim, …])

    Plot a single bycycle fit using plotly.

    plot_bg(dfs_features, sigs, fs[, titles, …])

    Plot 2D bycycle results.

    plot_bgs(dfs_features, sigs, fs[, xlim])

    Plot 3D bycycle results.

    Reports¶

    Generate analysis html reports.

    generate_report(output_dir[, fms, bms, …])

    Generate all FOOOF and/or Bycycle reports.

    generate_header(report_type, output_dir, dtype)

    Include masthead and subject info in a HTML string.

    generate_fooof_report(model, fooof_graphs, …)

    Include fooof settings, results, and plots in a HTML string.

    generate_bycycle_report(fit_kwargs, graph, …)

    Include bycycle settings, results, and plots in a HTML string.

    Input/Output¶

    Check input and output directories and model saving.

    check_dirs(input_dir, output_dir)

    Check input/out directories.

    clean_mkdir(dir_path)

    Make new directories and deleting pre-existing.

    save_fooof(model, output_dir)

    Make output directories and save FOOOF fits.

    save_bycycle(dfs_features, output_dir)

    Make output directories and save bycycle dataframes.
